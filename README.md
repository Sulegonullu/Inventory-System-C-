# Inventory System (C++)

## English

A simple inventory management system developed in C++ using Object-Oriented Programming principles.

### Features

* Add Item
* Remove Item
* Use Item
* Search Item
* Display Inventory
* Save Inventory to File
* Load Inventory from File
* Item Type System
* Item Rarity System
* Inventory Capacity Limit

### Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Vector
* File Handling (fstream)
